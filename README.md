# Coffee Cooperative Exercise

Este proyecto es una plataforma que conecta a **Baristas** con **Cultivadores de Café**. Su objetivo principal es permitir que los baristas sean notificados automáticamente cada vez que aparecen nuevos lotes de café disponibles. Para la realización de este proyecto se implementó el patrón *Observer*,  permitiendo que los objetos (baristas) se suscriban y reaccionen a eventos (nuevos lotes) generados por otro objeto (cultivador).

La arquitectura de este proyecto se basa en una implementación previa exitosa de este mismo patrón, lo cual facilitó el desarrollo y la estructura del código actual.
